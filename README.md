This repository contains the exported Azure Resource Manager (ARM) template and parameters for the infrastructure deployed during Lab 1. This template captures the following resources:

Virtual Machine: An Ubuntu 22.04 LTS instance named WEB1.

Web Server: A pre-configured Apache2 service.

Networking: Includes a Virtual Network (VNET-LAB1), a subnet (SUBNET-WEB), and a Network Security Group (NSG-WEB).

Storage: A 16 GiB Standard SSD managed data disk mounted at /data.
